About this product

  Libre Organización: a proposal's archetype.
    
Installation

  * Tested on Plone 3.3.4

  * Copy the collective.libreorganizacion folder to your buildout src folder and add the egg to your buildout.cfg eggs and zcml sections.
  
  * Restart Zope
  
  * Go to Plone configuration menu "Add/Remove product", select the product and install it.
  
  * Have fun!

Testing

  * run 'bin/buildout -s collective.libreorganizacion'
